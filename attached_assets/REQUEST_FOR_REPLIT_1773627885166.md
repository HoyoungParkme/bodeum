# Request For Replit

## 작업 요청

이 프로젝트의 현재 목표는 `디자인 프로토타입 고도화`입니다.

현재 제품의 핵심 흐름은 아래와 같습니다.

1. 랜딩
2. 분석 시작
3. 자기 분석 / 타인 분석 선택
4. 100문항 설문
5. 챗봇 심화 질문
6. 최종 리포트

이 흐름을 유지한 상태에서 디자인과 UX를 재구성해 주세요.

## 꼭 반영해 주세요

- Big Five 기반 서비스라는 점
- 설문이 길어도 덜 지치게 보이도록 설계
- 챗봇이 따뜻한 코치형으로 느껴지도록 설계
- 최종 리포트는 읽는 재미보다 `이해하기 쉬움`이 우선
- 모바일에서 무너지지 않을 것

## 결과 화면에서 중요하게 보여야 하는 것

- 성향 한 줄 요약
- Big Five 레이더 차트
- 강점
- 주의 포인트
- 소통 스타일
- 코칭 팁
- 왜 이렇게 나왔는지에 대한 근거

## 원하는 화면 리스트

- Landing
- Assessment Type Select
- Invite / Consent
- Survey Intro
- Survey Progress
- Chat Interview
- Final Report

## 이번 단계에서 괜찮은 가정

- 데이터는 목업으로 처리해도 됨
- API 호출은 가짜 데이터여도 됨
- 실제 분석 로직은 없어도 됨

## 중요한 톤

- 따뜻함
- 차분함
- 신뢰감
- 과장되지 않은 자신감

## 현재 상태

- 현재 코드에는 React + Vite 기반 화면 뼈대가 있음
- API는 최소 요약 엔드포인트만 존재함
- 디자인은 구조 확인용 수준이므로 자유롭게 재구성 가능
